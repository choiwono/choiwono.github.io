## 자주 헤깔리는 mac 단축키

## mac 소스 네비게이션 단축키
- 해당 소스의 hierarchy : command + t 또는 fn + f4
- Ctrl + 마우스커서(혹은 F3) : 클래스나 메소드 혹은 멤버를 상세하게 검색하고자 할때
- Alt + ->, Alt + <- : 이후, 이전
- Ctrl + o : 해당 소스의 메소드 리스트를 확인하려 할때
- F4 : 클래스명을 선택하고 누르면 해당 클래스의 Hierarchy 를 볼 수 있다.
- Ctrl + Shift + R : 파일 쉽게 열기 (파일명으로 검색)
